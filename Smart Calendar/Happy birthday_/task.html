Description

It may happen that you do not have time or forget to wish some of your friends a happy birthday. Social networks help you a lot in this! They have functionality that reminds you of your friend's birthdays. Let's try to teach your calendar to do something similar.

Your calendar is already pretty good at working with notes, and that's great! Teach your program to save the dates of birthdays of your friends, acquaintances, and relatives, and it will remind you of these dates!

Please note that the events in the files are removed before each test run for the tests to work correctly.

Objectives

Let's improve the program! After updating the notes of the calendar, ask for birthdays. Similarly to dates, you can add or delete birthdays. To do this, you first ask the user to enter the person's name, and then his date of birth. After this, you will also print in birthday notifications how old the person would be.

  1. Display all current notes after displaying the current date and time. If no current notes, you may skip this step.
  2. Ask the user how many notes they want to add or delete. Then you read the number. A positive number means that the user wants to add several notes, and negative means that the user decides to delete that many notes.
  3. To add notes, read data and text and save the result in a file.
  4. To delete a note, ask the user to enter the text of the note and remove information from your file. If such a note is not found, you need to print an error message and ask again
  5. Ask the user how many birthdays they want to add or delete. Then you read the number. A positive number means that the user wants to add several birthdays, and a negative one means that the user decides to delete that many birthdays.
  6. To add birthdays, read the name and birth date and save the result in a file.
  7. To delete birthdays, ask the user to enter the name and remove information from your file. If such a person is not found, you need to print an error message and ask again
  8. In the end, print all saved notes and birthdays

Hint: you cannot save the birthday date that will be in the future. Also, you can use two different files for birthdays and events, it's up to you!
