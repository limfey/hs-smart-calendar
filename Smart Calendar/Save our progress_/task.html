Description

Your program can create a note and even calculates the remaining time before it occurs. However, you can see that by closing and restarting the program, you need to start all over again. It can be compared with a notebook where we write something but then tear out a sheet. Let's fix it! You will teach the program to save and display your notes on the screen.

Please note that the events in the files are removed before each test run for the tests to work correctly.

Objectives

In this stage, you need to save your events. The easiest way is to write your results to a file and then read the data from it in the next session.

  • Input and output are the same as in the previous step.
  • Save the information about your events (the text and their date and time) to the notes.txt file and display information about all the notes entered earlier at the end.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1: the first session

Current date and time:
2020-05-13 20:33:03.907875
Let's add a note to the calendar.
Enter year:
> 2020
Enter month:
> 06
Enter day:
> 20
Enter hour:
> 18
Enter minute:
> 00
Enter a text:
> Visit a doctor
Before the event note "Visit a doctor" remains:
37 day(s), 21 hour(s) and 26 minute(s).
