Description

The calendar is essential in our daily life if we must schedule events and know how much time is left before them. But the implementation of all the useful functions requires some time. Let's start with something simple and display the current date and time on the screen.

Objectives

In this stage, the program should output two lines:

  1. The phrase Current date and time;
  2. The current date and time in the datetime format.
Example

Example 1:

Current date and time:
2020-05-12 19:57:06.320378
