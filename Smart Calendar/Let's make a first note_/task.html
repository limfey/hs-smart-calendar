Description

You learned how to recognize and display the date and time, and it's excellent! However, the calendar will become more interesting if you know how to compare specific dates and events. In this stage, let's try to create notes. You will teach your program to add a note and display it on the screen.

Objectives

In this stage, your program should:

  1. Display the current date and time;
  2. Get the date and time of the event from a user. Your program should ask the user to input a year, month, day, hour, and minute separately, one by one (look at the example below). Each user input will contain one number as an answer;
  3. Ask the user to enter the text of the note;
  4. Display how much time is left before the action described in the note.

Output the message on every step! You can see all messages in the example below.

Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Current date and time:
2020-05-13 20:33:03.907875
Let's add a note to the calendar.
Enter year:
> 2020
Enter month:
> 06
Enter day:
> 20
Enter hour:
> 18
Enter minute:
> 00
Enter a text:
> Visit a doctor
Before the event note "Visit a doctor" remains:
37 day(s), 21 hour(s) and 26 minute(s).
